# Parsers

## Normalizing properties
Parsers are, among other things, responsible of converting properties to the Kanttiinit standard™️:

|Property|Meaning|
|--------|-------|
|A|contains allergens|
|E|egg-free|
|G|gluten-free|
|K|vegetarian|
|L|lactose-free|
|M|milk-free|
|N|contains nuts|
|S|soy-free|
|SE|contains celery|
|SO|contains soy|
|T|healthier choice|
|V|vegan|
|VL|low in lactose|
|VS|contains garlic|
