# Parsers

## Normalizing properties
Parsers are, among other things, responsible of converting properties to the Kanttiinit standard™️:

|Property|Meaning|
|--------|-------|
|A*|contains allergens|
|C*|contains celery|
|E|egg-free|
|G|gluten-free|
|H|healthier choice|
|L|lactose-free|
|LL|low in lactose|
|M|milk-free|
|N*|contains nuts|
|O*|contains garlic|
|S|soy-free|
|S*|contains soy|
|V|vegetarian|
|VV|vegan|
